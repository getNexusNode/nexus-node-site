Google Contacts Import

Your existing contacts,
in one click.

Already have hundreds of contacts in Google? Import them all via vCard export. Names, emails, phones, addresses — everything lands in NEXUS NODE automatically.

1

Export from Google

Go to Google Contacts → Export → select vCard (.vcf) format → download the file.

2

Import into NEXUS NODE

Open NEXUS NODE → Contacts → Import → select your .vcf file → done.

3

All data lands automatically

Every contact appears with their full info. Enrich with photos, categories and warmth scores from there.
